ecert

  • Tech Stack: HTML, CSS, Tailwind, React, Next.js, PHP, Laravel, Stripe, Xero Integration, Data Migration
  • Project Link: ecert

eCert is an online platform that provides electronic certificates, primarily offering healthcare courses. The platform supports two types of users: individuals and companies. Companies can choose between two packages—credits or user-based—and, upon successful signup, can create and assign course packs to invited individuals. Both individual and company users can take tests for the courses, and upon successful completion, they receive electronic certificates.

eCert integrates Stripe for subscription management and recurring payments. Loqate API is used for address validation, and Xero integration ensures seamless financial record-keeping. A major component of the project was data migration, transitioning the platform from an older version of Symfony with MySQL to a new database structure using the latest MySQL version, which was a significant challenge.

Key Points

  • Led the migration of a legacy Symfony project to a new, scalable Laravel application, optimizing the MySQL database schema and migrating 500+ companies and 50,000+ users seamlessly.
  • Integrated AWS services for scalable hosting, ensuring high availability and performance across platforms.
  • Designed and developed a robust API for certificate management, resulting in the issuance of 1 million+ certificates.
  • Utilized Stripe for payment processing and Geckoboard for real-time stats tracking to provide actionable insights to the system administrators.
  • Improved system performance and data handling by optimizing database queries, reducing migration time by 50% and ensuring smooth operation across all tenants.
