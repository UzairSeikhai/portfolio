HealthCareTalentLink (HCTL)

  • Tech Stack: HTML, CSS, React, Next.js, PHP, Laravel, Stripe, Google Location API, AWS EC2, AWS S3
  • Project Link: HCTL

HealthcareTalentLink (HCTL) is a professional platform that connects the healthcare community with its talent, bridging services, products, professionals, and opportunities within the healthcare industry. The platform offers various modules including service management, hiring management, live interview services, and ads management. Users can create multiple companies, post jobs, hire candidates with a thorough screening process, and conduct live interviews. HCTL also enables individuals to provide and avail services through their profiles. Integration with Stripe facilitates payments, while Google Location API enhances location-based features. The platform is hosted on AWS EC2 with AWS S3 used for secure data storage.
