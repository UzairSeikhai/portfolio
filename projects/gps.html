GPS

  • Tech Stack: HTML, CSS, Bootstrap, jQuery, PHP, Laravel, REST APIs, Stripe, PayPal, Amieos
  • Project Link: GPS

GPS is a web-based application developed for the former president of Ivory Coast, focusing on profile and event management. This application features a custom CMS, allowing for easy management of all pages through an admin panel. After creating the custom CMS, the client requested an e-commerce feature to be implemented within a short timeframe. We utilized the Laravel Amieos e-commerce package, working to stabilize it post-integration before delivering a successful solution.

Additionally, we developed REST APIs to support a mobile application, and the platform is multilingual, catering to French-speaking users with support for two languages. The application also includes a donation feature, allowing users to make contributions through PayPal.
