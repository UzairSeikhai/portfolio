ClosetKeeper

  • Tech Stack: HTML, CSS, Bootstrap, jQuery, Livewire, Laravel, Stripe
  • Project Link: ClosetKeeper

Closetkeeper is an online platform for laundry services, where two types of users can register: customers and service providers, each with separate dashboards. A super admin dashboard oversees the operations of both user types, including managing disputes between them. The platform integrates Stripe for recurring payments and offers a manual payment option via cheques. If payments are not received, users can be temporarily frozen, restricting their actions until dues are cleared. Livewire is used for real-time rendering of certain elements, enhancing user interaction and responsiveness throughout the application.
